# Task 1 - Network Scanning using Nmap

## Objective
Perform host discovery and service enumeration using Nmap.

## Commands Used

nmap -sn 192.168.29.0/24

nmap -Pn -sV 192.168.29.66

## Deliverables
- Scan results
- Screenshots
- Analysis of open ports

## Outcome
Successfully identified active hosts and enumerated running services.
